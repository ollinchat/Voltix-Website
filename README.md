# Najami · Zatout — Industrial EPC Joint Venture

Heavy industrial luxury corporate website for the Sami Najami & C. Zatout joint venture.

## Stack

- React 19 + TypeScript
- Tailwind CSS (Midnight Slate + Steel palette)
- Framer Motion (mechanical transitions)
- Full EN/HE bilingual with RTL support

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Routes

| Path | Page |
|---|---|
| `/` | Main site |
| `/legal` or `/terms` | Legal & Terms of Service (EN/HE, RTL) |

## Key Features

- **JV Synergy** — Combined Power metrics: 82,000 sqm factories, 15,000 tons steel, full EPC
- **Legal Page** — Full Terms of Engagement with IP, liability, JV, HSE, and governing law sections
- **Services** — Interactive EPC, Iron Construction, Pipeline cards
- **Portfolio** — Filterable Energy / Steel / Water categories
- **Contact** — Responsive form with custom toast notification
- **Design** — Blueprint grids, razor-sharp edges, glassmorphism, command-center aesthetic
